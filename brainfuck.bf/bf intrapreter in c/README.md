## About
Brainfuck-C is a [Brainfuck](http://en.wikipedia.org/wiki/Brainfuck) interpreter written in C.

## Usage
Compile using gcc and run: gcc brainfuck -o bf
```
./bf <filename>
```

## Examples
```
./bf hw.bf
```
